The Loxberry extends your Loxone Smarthome by many interesting and useful features, which can not provided by the Loxone-System itself. These features are realized using Plugins, which can be installed later on.

The Basic System of the Loxberry is free of charge and is maintained by volunteers in their leisure time. The Source Code is free and is distributed under the Apache 2.0-License. The Loxberry arose from an idea in the Loxone-Forum (https://www.loxforum.com/).

If you like and use the the LoxBerry or maybe use it in comercial circumstances, please think about a donation. We, the developer of the LoxBerry, do not want money. But there are many people who are not as fortunate as us. And these people need our support. Even a little amount helps, to help these people. A donation is quickly done and possible via money transfer, credit card or Paypal:

Donate for the German Child cancer aid society

In the following Setup-Wizard the basic System Setting like Networking or Time Server are done. The Loxberry will not work without these mandatory settings. All further Plugins (Features) can be installed and configured later from the Main Menu.
