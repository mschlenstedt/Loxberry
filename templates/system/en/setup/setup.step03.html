3rd Step: Network Settings

This step configures the Network Settings of your Loxberry. The Loxberry can communicate via cable (LAN) or Wireless (WLAN) using a supported USB-WLAN-Adapter connected to your network. In case of a WLAN-Adapter to be used, the device must be plugged into a free Loxberry USB-Port now.

If you use an Internet Router, the Network Settings are usually done via DHCP-Protocol automatically by the Router to the Loxberry. This is useful but it can happen, that a DHCP Client changes his IP-Address in the future. But it is very important, that the Loxone Miniserver and the Loxberry have both always their own and same IP-Address all the time. If not, the communication between them is not possible. Due to that we strongly recommend to configure the DHCP Settings in your Router or DHCP Server to assign always the same IP Address.

If this feature is not supported by your Router, the Loxberrxy (and the Loxone Miniserver) must be configured manually with a static IP-Address. To ensure that no other device uses these IP-Addresses the Addresses should be outside of the DHCP Range.
