1st Step: Define Administrator

Your first job is to define an Administrator-Account. This Account allows you to manage the settings of your Loxberry. You can enter an username you like (e. g. "michael"). A good password should have 12 characters minimum. Please use upper- and Lowercase characters and digits. Special characters can improve the password security - but not all are allowed - use the hint near the input boxes to get a clue what's allowed.

  Allowed characters: A-Z a-z 0-9
  Allowed characters: A-Z a-z 0-9 -_,.;:!?&()?+§%=
  Please enter the Password again to prevent typing issues
