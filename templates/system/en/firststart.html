First Start?

It seems you started the Loxberry the very first time. We recommend to use the Setup-Wizard for the Basic Installation of the System. The Wizard will guide you through the most important, mandatory settings of the Loxberry (e. g. the Network-Configuration). The Wizard needs to be executed one time only.

Experienced Users can skip the Wizard of course. The Setting can be done in Main Menu of the Loxberry also. In addition all settings can be done on the Linux-Commandline, if needed.

Browser Support: We try to make LoxBerry's webinterface as compatible as possible. Unfortunately especially the Microsoft Browsers IE and Edge do not support all standards completely and correctly. We suggest to use Chrome or Firefox with your LoxBerry. If you end up in any problems regarding the webinterface, please try one of these browsers first.
