Use Cloud-DNS: Check the checkbox to use the Loxone Cloud-DNS-Service to reach the Miniserver from extern.

IP-Address or Hostname: The IP-Address or Hostname of the Loxone Miniserver. Please ensure that the Miniserver is reachable all the time using this IP/Name.

Portnumber: The Portnumberto reach the Webinterface of the Miniservers. Usually Port 80.

MAC-Address: The MAC-Adresse of the Miniserver if you use the Loxone Cloud DNS Service. You can find the MAC-Address in the Adminarea of the Miniserver or in the Loxone Config. It's a 12 characters long string starting with e.g. EEE000xxxxxx or 504Fxxxxxxxx.

Public FTP-Port: The public FTP-Port of the Miniserver if you use the Loxone Cloud DNS Service. You can manage this Port in your Router (e.g. Fritz!Box). The intern Port should be kept on 21. Otherwise the Fritz!Box Firewall / Application Layer Gateway can not identify the traffic as FTP-Traffic. The same can happen for other Router Models.

Miniserver Username: Username of an Administrator-Account of the Miniserver. Often "admin".

Miniserver Password: The Password associated to the above Administrator-Account.

Note / Link: A field for free usage. e. g. Customername, Location, ....
If you enter a Link (URL), an Open-Button is automatically displayed to open the Link in a new Window.
