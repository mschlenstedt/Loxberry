Inputs / Selects

Core: input.lb-input for single-line text fields.
Core: input.lb-input[placeholder] uses --lb-input-placeholder-text and --lb-input-placeholder-opacity.
Core: input.lb-input[type=time] with data-role="none" for the native browser time picker.
Core: .lb-form-row.lb-disabled dims a complete form row.
Core: input.lb-input[disabled] uses the disabled tokens of the input component.
Core: select.lb-select and data-role="none" for a native dropdown.
Note: select has no real placeholder. The clean Core pattern is a disabled first option[disabled][selected].
Core: textarea.lb-textarea. Placeholder also uses textarea::placeholder.
