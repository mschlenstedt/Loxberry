Dieses Plugin hat keine Konfigurationseinstellungen.

Diese Seite kann dir aber helfen, den richtigen URL für die Loxone Config Parameter der Loxone Intercom zu finden.

Dieses Beispiel ist für eine Trendnet TV-IP310PI gültig.
   
   
   
Bitte trage in der Loxone Config im Loxone Intercom-Objekt bei "Bild bei Klingeln (intern)" folgenden String ein:

http://?kamera=Kamera-Hostname:80/Streaming/channels/1/picture&user=Benutzername&pass=Passwort Testen
